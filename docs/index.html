About this Project

This project encompasses a set of Google Apps Scripts designed to automate various tasks involving Google Sheets, Drive, and Gmail. The scripts primarily focus on file management, information retrieval, and automated email creation. These functionalities aid in organizing and sharing documents efficiently, particularly in educational or collaborative settings.

  • File Management: Scripts to create and manage files based on data from Google Sheets. This includes copying files with specific names, gathering file information, and setting sharing settings.
  • Email Automation: Scripts to create draft emails in Gmail, tailored for specific recipients, without sending notifications for shared files.

Through the automation of these tasks, the scripts streamline the processes involved in managing and distributing files, as well as communicating information related to these files.

Prerequisites

  • A Google account with access to Google Sheets, Drive, and Gmail.
  • Understanding of basic operations in Google Sheets, Drive, and Gmail.
  • Familiarity with Google Apps Script environment and JavaScript.

Setup

  1. Open Your Google Sheet: Access Sample Google Sheet.
  2. Copy the Google Sheet to make your sheet.
  3. Conduct GAS Authorization: Access `Initial Setting` Sheet and click the initial setting button. This enables you to go to the authorization page for Google Apps Script.
  4. Customize Constant Variables for Built-in Functions: Navigate to the Apps Script page and adjust the constant variables in `variables.gs` to suit your needs.

Features

  • File Copying and Naming: Automatically copies a template file and names each copy based on a list of names from a Google Sheet.
  • File Information Retrieval: Gathers and displays information about files in a specified folder, such as file name, URL, and sharing details.
  • Email Draft Creation: Creates personalized draft emails in Gmail for each record in a dataset, suitable for student and instructor communication.
  • Sharing Management: Sets and resets sharing permissions on files without sending notification emails.

Usage

Each function is available from the 'Custom Menu'.

  • Set information for student and faculty:
    1. Set information in the orange range in `system-data` sheet, which is the reference data for other sheets.
  • Copy and Name Files:
    1. Set the file name information in the orange range in `create-file` sheet.
    2. Click the `Copy and Name Files` to create copies of a specified file for each entry in the list from the Google Sheet.

      3. *You will be asked to put the URL of the target folder to store newly created files and the URL of the template file.

  • Retrieve and Display File Information:
    1. Click `Output File Info` to display detailed information about files in a designated folder in `share-file` (Column A - D).

      *You will be asked to put the URL of the target folder where files are stored.

      *Since the file has not been shared with anyone at this moment (unless you share the stored folder with someone), "Editors' emails" returns empty.

  • Manage File Sharing:

    Click either `Share Files Without Notification` or `Reset All Sharing Status` to manage file sharing settings.

    1. Share Files Without Notification: This function allows you to share files with specified users without sending them a notification. Necessary information such as email addresses of the users and the file information is automatically displayed by Google Sheet built-in formula.

      *You will be asked to put the URL of the target folder where files are stored.

      *Once you execute `Output File Info` again, the column, "Editors' emails" are supposed to show editors' emails.

    2. Reset All Sharing Status: This function resets the sharing settings of all files in a specific folder. It removes all users from the shared list and makes the files private.
    • Create Email Drafts:
    1. CLick `Create Drafts` to generate personalized draft emails in Gmail.

Others

  • Customization: Scripts can be customized according to specific needs. Users can modify the code in the Script Editor to fit their particular requirements for file management and email automation.
  • Debugging and Maintenance: Regular maintenance and updates might be required to ensure compatibility with the latest Google services updates. Users should monitor the scripts for any errors and make necessary adjustments.
  • Enhanced Functionality: Users with advanced scripting skills can expand the capabilities of these scripts by integrating additional Google services or implementing more complex logic.

Overall, this project provides a foundation for automating repetitive tasks in file management and communication, allowing users to focus on more critical aspects of their work or study. By leveraging the power of Google Apps Script, this solution offers a customizable and scalable approach to managing digital workflows.
